# Ẩm Thực / Recipe AI

Full-stack: Angular frontend + .NET 8 backend (Recipe AI Web).

## Cấu trúc

- **frontend/** – Angular (recipe-ai-web), port 4200
- **backend/**  – .NET 8 (RecipeAi.sln), API port 5080

## Chạy nhanh

**Backend**
```bash
cd backend && dotnet restore && dotnet run --project recipe_ai.app.web
```

**Frontend**
```bash
cd frontend && npm install && npm start
```

Xem chi tiết trong `frontend/README.md` và `backend/README.md`.
